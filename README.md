# Windows平台运维工具

## IIS6一键安装程序.bat

>一键安装iis6

### 用法

该bat文件放置于i386目录里。双击即可。或者可以与挑选出来的iis6安装文件放置同级目录。压缩后制作成自解压安装程序。

## install_msu_hotfix.bat

>用来批量安装msu格式补丁。

### 用法

将该bat文件放置与msu同级目录即可

## win10输入法输入不了汉字修复

>win10系统，经常会遇到输入不了汉字的情况，运行一下就可以了。

### 用法

直接双击执行即可

## 批处理运行ps1.bat

>直接双击就可以执行powershell脚本。

### 用法

替换里面的ps1名字为你脚本的名字。管理员执行即可
